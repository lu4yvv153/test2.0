marcas de linhas e paragrafos

Este texto mostra a atuação dos comandos de quebra de linhas e parágrafos.

Uma página HTML não reconhece o fim de uma
linha com o pressionamento da tecla Enter.
Para quebrar uma linha e continuar imprimindo o texto na linha seguinte
Utilize o comando "BR".

Já o comando "P", será interpretado pelo browse como um parágrafo,
ou seja, antes de imprimir o texto digitado após o comando "P" será dado um salto de linha.
